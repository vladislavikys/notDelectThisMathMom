# notDelectThisMathMom

🧮 Игра-тренажёр по арифметике для детей с защитой доступа по личным кодам.

## Файлы
- **index.html** — игра для учеников (вход по коду): сложение, вычитание, умножение, деление; выбор количества примеров и диапазона чисел; 2 попытки на ответ, показ верного ответа, история и процент в конце; настройки учителя (звук, показ ответа, повтор правильного, отрицательные числа).
- **teacher.html** — панель учителя: создание/отзыв кодов, срок действия, лимит устройств, статистика.

## Доступ учеников
Доступ ограничен личными кодами (по числу оплат) через Supabase:
- код привязывается к устройству ученика (нельзя раздать ссылку всем);
- лимит устройств на код, срок действия, мгновенный отзыв;
- вся проверка — на сервере (Edge Functions), из браузера обойти нельзя.

> Секретный ключ учителя и service-ключи **не** хранятся в этом репозитории.
> В коде есть только публичный (publishable) ключ Supabase — он безопасен для публикации.

## Запуск
Открыть `index.html` в браузере или выложить на статический хостинг (Netlify / Cloudflare Pages / GitHub Pages). Панель `teacher.html` держать отдельно от учеников.
